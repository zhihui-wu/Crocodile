## Git Commit Message规范：

- FEAT 增删改一个功能

- FIX 修复一个Bug

- DOCS 增删改文档

- STYLE 代码美化或者其它不影响功能的代码修改

- REFACTOR 重构一个功能

- PERF 优化一个功能

- TEST 增删改一个单元测试

- CHORE 框架升级、Jar包引用增删改等
